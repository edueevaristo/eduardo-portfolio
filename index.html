Sobre mim

Meu nome é Eduardo Evaristo, tenho 22 anos e atuo na área de desenvolvimento há 1 ano e 11 meses.
Sou apaixonado por tecnologia! Tive meu primeiro contato com a informática aos 15 anos de idade e desde então venho descobrindo um pouco mais sobre essa área maravilhosa. Tenho conhecimento em: JavaScript/JQuery, SQL, MySQL, API Rest, HTML, CSS, Bootstrap, GIT, PHP, CodeIgniter, Laravel e atualmente, estudando Python e React Native.
Atualmente, trabalho como Desenvolvedor Web Júnior na Fábrica de Códigos onde tenho contato direto com as linguagens,frameworks e banco de dados: PHP (CodeIgniter 3), SQL (PostgreSQL, Redis), HTML, CSS (Bootstrap 3), Javascript (Puro, JQuery), Golang. A Fábrica de Códigos foi o meu primeiro emprego como programador, antes da mesma, atuei como Analista de Vendas na Connect Parts por 1 ano e 8 meses.
